### Controls
- ➡️ : move paddle right
- ⬅️ : move paddle left
- 🖱  : If the mouse is in playing area, the paddle follows it. 

### Mission
- You win if enemy lost all his lives
- You lose if you lost your lives
- Lives is reduced if the ball flies for a paddle.
